HTML5 e CSS

HTML5

O HTML5 é a mais recente versão da linguagem de marcação HTML (Hypertext Markup Language), utilizada para desenvolver e estruturar páginas da web. Lançado em 2014, o HTML5 trouxe diversas melhorias e novidades em relação às suas versões anteriores, permitindo a criação de sites mais dinâmicos e interativos.

Elementos semânticos HTML5 Usei, sim(👍🏻) ou não(👎🏻)? Breve descrição
div 👍🏻 Usei o div para agrupar conteúdo, no caso agrupar ou um conjunto de opções ou opções uma a uma. Usei-o por exemplo para as perguntas de opção do quizz, como por exemplo a pergunta ❷ e ❸.
h1...h6 👍🏻 Utilizei o h1...h6 de forma a organizar o meu conteúdo, separando-o, por exemplo. É utilizado maioritariamente para criar um titulo ou um subtitulo de uma secção, tal como acontece no quizz, na introdução, entre outros.
header 👍🏻 Utilizei o header de forma a indicar os meus cabeçalhos, o que acontece em todas as secções, onde coloco o nome da cidade e uma imagem da mesma.
nav 👍🏻 Utilizei o nav para representar as minhas secções de navegação em uma ou mais página`s web. Este contém links para outras minhas páginas, de forma a que o utilizador do site clice nos mesmos e seja direcionado para o local que deseja.
main 👍🏻 Utilizei o main de forma a indicar o conteúdo principal e único, que não se repete. Usu-o em todas as paginas dentro do elemento body de forma a indicar que aquele conteúdo só está presente exclusivamente naquela página/secção.
article 👍🏻 Utilizei o article de forma a indicar uma porção do documento que pode ser vista de forma independente, usu-o por exemplo nesta página quando dou uma breve introdução do que é o HTML5 e apresento uma tabela sobre HTML5 e o mesmo acontece mais abaixo para o CSS.
section 👍🏻 Utilizei section`s de forma a dividir os conteúdos por exemplo da página da localização, em que uma section é um breve texto de onde a minha cidade se situa e a outra section é o mapa do google maps.
aside 👍🏻 Utilizei aside de forma a deixar uma sugestão/opinião no final do vídeo que coloquei na Multimedia.
footer 👍🏻 Utilizei footer`s para indicar o rodapé das minhas secções, utilizei em todas as páginas de forma a criar um copyright.
details 👍🏻 Utilizei details de forma a adicionar detalhes adicionais na pagina de Introdução podendo o utilizador ver ou esconder. Estes estão contidos num elemento summary.
meter 👍🏻 Utilizei meter para definir uma medida escalar estática dentro de um intervalo usando a vontade que um utilizador terá antes de visitar Cancún sem ter visto o meu website e a vontade do utilizador visitar Cancún depois de visitar o meu website, por completo.
progress 👍🏻 Utilizei progress para definir uma barra de progresso do meu quizz, onde o utilizador pode ver a que percentagem de progresso vai.
abbr 👍🏻 Utilizei abbr para indicar o significado da palavra XCARET, na minha página da multimédia.
address 👍🏻 Utilizei address no final do meu quizz de forma a deixar uma informação sobre como o utilizador me pode contactar, neste caso por email.

CSS

O CSS (Cascading Style Sheets) é uma das principais linguagens de estilo utilizadas para estilizar páginas web. Com o CSS, é possível definir a aparência visual de um site, incluindo o layout, as cores, as fontes, as animações e muitos outros aspectos visuais. Ao separar a estrutura do conteúdo da aparência visual, o CSS permite que os desenvolvedores criem sites mais eficientes e flexíveis. Além disso, o CSS permite que os sites sejam facilmente personalizados e atualizados sem afetar a estrutura do conteúdo. Com sua ampla gama de recursos e ferramentas, o CSS é essencial para qualquer desenvolvedor web que deseja criar páginas web visualmente atraentes e profissionais.

Tipos de Seletores Usei, sim(👍🏻) ou não(👎🏻)? Breve descrição
universal 👍🏻 Utilizei o seletor universal de forma a mudar a fonte do texto de todo o meu site.
tipo 👍🏻 Utilizei o seletor tipo de forma a mudar a cor do meu h1...h6 para branco.
classe 👎🏻 Não utilizei o seletor classe porém este serve para aplicar estilos a um ou mais elementos HTML que compartilham o mesmo valor de classe. O setor da classe começa com um ponto (.) seguido pelo valor da classe, permitindo aplicar estilos específicos a grupos de elementos.
id 👎🏻 Não utilizei o seletor id porém este serve para selecionar elementos HTML específicos e aplicar estilos a eles. Ele é fundamental para estilizar uma página da web de forma consistente e eficiente.
atributo 👎🏻 Não utilizei o seletor atributo porém este serve para selecionar elementos HTML que possuem um atributo especifico e aplicar estilos a eles. Ele é útil para estilizar elementos com base em seus atributos, como o valor de um atributo href em um elemento a.
pseudo-classe 👍🏻 Utilizei o seletor pseudo-classe para mudar a cor das colunas dos meus tr, caso a linha da coluna for par terá uma cor, caso for ímpar terá uma outra cor diferente.
pseudo-elemento 👍🏻 Utilizei o seletor pseudo-elemento para colocar no meu meno um icone antes e depois da sua hiperligação, de forma a estiliza-lo.

Combinadores de Seletores Usei, sim(👍🏻) ou não(👎🏻)? Breve descrição
descendentes 👎🏻 Não utilizei os combinadores de seletores descendentes porém estes servem para corresponder a qualquer elemento f que seja um descendente, ou seja, esteja "dentro" de um elemento E.
filhos 👎🏻 Não utilizei os combinadores de seletores filhos porém estes servem para corresponder quando um elemento F é precedido por um elemento E (não necessariamente imediato, como no caso de E > F)
adjacentes 👎🏻 Não utilizei os combinadores de seletores adjacentes porém estes servem para selecionar elementos que aparecem imediatamente após outro elemento específico. O combinador é representado pelo simbolo "+" e é colocado entre os seletores que se deseja combinar. Em suma, os combinadores de seletores adjacentes são uteis para selecionar elementos que estão diretamente relacionados a outros elementos da HTML e aplicar estilos específicos a eles.
imediatamente adjacente 👎🏻 Não utilizei os combinadores de seletores imediatamente adjacentes porém estes são uteis para selecionar elementos que estão diretamente relacionados a outros elementos no HTML e aplicar estilos especificos a eles, mas somente se estiverem ao mesmo nivel hierárquico.
